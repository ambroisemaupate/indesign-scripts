# Markdown script for InDesign
---------------------------------------------------
# Installation

1. Download the markdown.jsx file
2. Launch InDesign app
3. Open Scripts panel (Window > Utilities > Scripts)
4. Choose User folder and click on "Reveal in Finder" (Option menu of the scripts panel)

5. Copy the downloaded .jsx file to the folder Scripts Panel
6. The script should appear in InDesign Scripts panel.

---------------------------------------------------
# Usage

1. Create a new document
2. Paste some Markdown in your document
3. Double click on the markdown.jsx script (Script Panel)
5. Choose the selection (Document or selected story)
4. Connect each markdown entity to a predefined style (Paragraph or Character)
5. Go !
